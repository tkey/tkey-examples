# Torus JS Custom JWT

## How to Use

### Download Manually

```bash
npx degit tkey/tkey-examples/torus-js/torus-js-custom-jwt torus-js-custom-jwt
```

Install & Run:

```bash
cd torus-js-custom-jwt
npm install
npm run start
# or
cd torus-js-custom-jwt
yarn
yarn start
```
